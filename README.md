# Weather-Forecast-Cluj-Napoca
Weather category forecast project, avaialble at: https://huggingface.co/datasets/LaurentiuStancioiu/Cluj-Napoca-Weather-OpenWeather-data#cluj-napoca-weather-dataset

I have built models, inclusing Logostic regression, k-Nearest Neighbors (k-NN), Decision Tree, Random Forest Classifier, Neural Network (MLPClassifier with Hyperparameter tuning.
